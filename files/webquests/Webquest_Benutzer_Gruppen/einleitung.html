Einleitung

Linux ist ein Mehrbenutzer-System, das heißt es kann von mehreren Benutzern gleichzeitig genutzt werden. Dieses Prinzip dient der Teilung von Ressourcen.
Sobald Speicherplatz und Dienste auf zentralen Computern zur Verfügung stehen, sollen auch mehrere Nutzer darauf zugreifen können. Beispielsweise kann ein System in einem Betrieb zwischen verschiedenen Benutzern der Abteilungen unterscheiden und entsprechende Rechte zuordnen.
Auch für den Heimcomputer macht dies Sinn. Alle Familienmitglieder können ein Linux-System benutzen und dennoch ihre privaten Daten darauf verwalten.
Eine Besonderheit stellt der Benutzer root dar. Er besitzt alle Privilegien, um das System zu administrieren.
In diesem WebQuest lernen Sie Benutzer und Gruppen zu erstellen und zu verwalten.
