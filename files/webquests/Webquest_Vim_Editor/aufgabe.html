Aufgabe

Der interne Mitarbeiter-Bereich auf dem Webserver Ihres Ausbildungsbetriebes soll einen simplen Zugriffsschutz erhalten. Sie erhalten daher die Aufgabe, eine entsprechende .htaccess - Datei zu erstellen. Sie haben ausschließlich über die Kommandozeile Zugriff auf den Webserver und daher mit dem Editor vim. Da Ihnen zunächst einige Angaben des Administrators fehlen, erstellen Sie erst einmal eine Vorlage. Sie schreiben folgenden Inhalt in die Datei /www/htdocs/intern/.htaccess :


	# .htaccess-Example
	AuthType Basic
	AuthName "Password Required"
	AuthUserFile /www/passwords/password.file
	Require User admin

Anschließend speichern Sie die Datei und schließen den Editor.
Nach kurzer Zeit erfahren Sie vom Administrator die Details. Sie öffnen die vorhandene Datei, speichern eine Sicherungskopie unter anderem Namen und ändern die Datei wie folgt ab:


	# .htaccess-Datei fuer Servicebereich
	AuthType Basic
	AuthName "Service-Bereich"
	AuthUserFile /usr/verwaltung/.htusers
	AuthGroupFile /usr/verwaltung/.htgroups
	Require Group Servicetechniker
	Require User Werner Dieter Heidi
