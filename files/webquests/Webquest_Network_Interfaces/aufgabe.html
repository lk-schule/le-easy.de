Aufgabe

Sie erhalten von Ihrem Betrieb IBL-Network-Consulting GmbH den Auftrag für den Kunden Scheuermann & Soehne AG ein Netzwerk einzurichten. Das Netzwerk besteht aus 1 Server und 3 Clients. Auf allen Rechnern ist ein Linux-System installiert und die Konfiguration der Netzwerk-Schnittstellen erfolgt über /etc/network/interfaces. Es gelten folgende Vorgaben für die Konfiguration:

    • Die zu verwendende Schnittstelle ist bei allen Rechnern Ethernet.
    • Das Interface des Servers wird beim Booten gestartet, die Interfaces der Clients sollen manuell gestartet werden können.
    • Als Protokoll wird IPv4 verwendet.
    • IP-Adressen werden statisch zugewiesen. Geben Sie Adresse, Netzmaske, DNS und das Gateway an.
  1. Richten Sie das Netzwerk gemäß den genannten Vorgaben ein.
    Erstellen Sie die Konfigurationsdateien für alle 4 Rechner.
  2. Starten und stoppen Sie die Schnittstellen der Clients manuell.
    Geben Sie die verwendeten Befehle an.
  3. Testen Sie die Verbindung zum Server mit dem Befehl ping.
  4. Worin liegen die Unterschiede zwischen einer manuelle Konfiguration über die interfaces-Datei und der Verwendung eines Networkmanagers (NM oder Wicd)?
  5. Auf welchen Systemen ist eine manuelle Konfiguration sinnvoll? Antwort bitte begründen!
  6. Auch mit dem Befehl ifconfig lassen sich Netzwerkschnittstellen konfigurieren. Wo aber liegt hier der Unterschied zu einer Konfiguration über /etc/ network/interfaces ?
