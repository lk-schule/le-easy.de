Aufgabe

Da Sie beim Einrichten von Servern immer wieder mühsam Benutzerkonten per Einzelbefehl einrichten müssen, wollen Sie sich die Arbeit mittels eines Skriptes erleichtern.
Schreiben Sie ein Skript, welches in einer Endlosschleife per Eingabe alle erforderlichen Daten zum Anlegen eines Kontos entgegennimmt und anschließend ein entprechendes Benutzerkonto anlegt.
Mit der Eingabe exit kann das Skript beendet werden. Wird das Skript mit dem Argument v gestartet, soll für jedes angelegte Konto eine Bildschirmausgabe mit Informationen über das ausgeführte Kommando erfolgen.
