Aufgabe

1)

Ihr Bekannter möchte seinen alten Computer komplett auf Open-Source umstellen. Da er noch sehr unerfahren auf diesem Gebiet ist, fragt er Sie nach Empfehlungen, was die Anwendungen betrifft. Er benötigt Anwendungen für die folgenden Bereiche:

  • Büro und Produktivität
  • Multimedia-Wiedergabe
  • Grafik-Bearbeitung
  • Email und Internet

Machen Sie Empfehlungen für jeden Bereich.

2)

Sie übernehmen für Ihren Fußballverein die Aufgabe, eine Website auf einem eigenem Server zu erstellen. Sie entscheiden sich, den Server ausschließlich mit freier Software zu betreiben. Sie benötigen Anwendungen für:

  • HTTP-Webserver
  • Content-Managment-System
  • Datenbank-Verwaltung

Informieren Sie sich, welche Anwendungen Sie dort einsetzen können.

3)

Ihr Bekannter richtet sich einen Server unter Linux ein. Da er kaum Erfahrung mit freier Software hat, fragt er Sie nach Rat. Sein Server soll folgende Funktionalitäten haben:

  • Fileserver
  • Mailserver
  • DHCP und DNS

Empfehlen Sie ihm entsprechende Open-Source-Software.

4)

Ihr Freund arbeitet als Anwendungsentwickler. Er hat bisher nur unter anderen Betriebssystemen entwickelt, möchte jetzt aber auf Linux umsteigen. Er bittet Sie daher um Empfehlungen für Enticklungsumgebungen und Werkzeuge. Er arbeitet mit folgenden Entwicklungssprachen:

  • C
  • Java
  • Perl
  • Python
  • PHP
