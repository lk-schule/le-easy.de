Analyse der Zugriffe auf einen Webserver

Nachdem Sie erfolgreich einen eigenen Webserver im Internet eingerichtet haben, finden Ihre Seiten eine fantastische Akzeptanz. Nun wollen Sie eine Analyse der Nutzung des Webservers erstellen. Hierzu bietet die Datei access.log des Apache-Webservers eine gute Möglichkeit.

In der Datei werden alle Zugriffe mit der IP-Adresse, dem Zeitpunkt des Zugriffs, das benutzte Betriebssystem, die entsprechende Webseite , etc. protokolliert.

Aufgabe:

  • Zählen Sie die Klicks, die in dieser Log-Datei verzeichnet sind?
  • Wieviel Klicks wurden an einem Tag auf Ihren Seiten verzeichnet?
  • Welche Klicks enthalten nicht das Wort karo-spiele?
  • Erstellen Sie eine Liste, wieviele Daten bei den einzelnen Zugriffen gesendet wurden.
  • Wieviel Klicks führten zu einer fehlerhaften Kommunikation?
